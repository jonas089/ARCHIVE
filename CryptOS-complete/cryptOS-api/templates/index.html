CryptOS platform ( CryOS ) ist eine seit März 2019 kontinuierlich entwickelte Cryptowährung welche nicht konzipiert ist herkömmliche Zahlungmethoden wie zum Beispiel den Euro oder den schweizer Franken im alltäglichen Gebrauch zu ersetzen.
Unser Ziel ist es stattdessen, mit Hilfe von einer öffentlich zugänglichen API möglichst einfachen und zuverlässigen Zugriff auf die "core – features" unserer Wallets und andere wichtige Komponenten zu offerieren.
Unser Fokus liegt auf Community, Entwicklung und vor allem mobiler Technologie für Android & IOS Geräte.

Features

Open Source

CryOS cryptocurrency is open source

SOON Github

Specification

Coins can not be mined
There is no POS
Masternodes are also not involved
Total supply : 4.000.000.000 (4 Billion)

Security

The PC core wallet is a fork of "Ravencoin" project
and running the X16s algorythm
( invented by "Pigeoncoin" ).
Sensitive user data is encrypted with SHA-256

Concept

CryOS wants to establish as a brand by combining mobile development and cryptocurrency.
We want to publish an official game in the future.

Coinsale

Our Coinsale is meant to fund future development
You can buy CryOS with Raven(Rvn) and Bitcoin(Btc) inside our mobile client
Presale prices : 1 Rvn = 250 CryOS
1 Btc = 5000000 CryOS


Coins available for the SALE : 250.000.000 (250 Million)

Centralisation

On launch the CryOS team has control of the whole supply
Over time coins will find their way to our community
Users can setup nodes and use our core wallet to decentralize their holdings
Except for the presale funds (250 Million coins) the whole supply is reserved for the community
It will be possible to earn coins by using our mobile wallet,
participating in contests and maybe at a later stage by using third party services
